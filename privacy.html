2. AI による録音の処理について

本アプリでは、録音音声を整理する目的で AI 処理を行います。具体的には、当方のサーバー（Supabase Edge Functions）を経由して、以下のとおり OpenAI（OpenAI, L.L.C.）の API へデータを送信します。

  • 音声の送信: 録音音声ファイルを OpenAI の音声文字起こし API（Whisper, whisper-1）へ送信し、テキスト化します。
  • テキストの送信: 文字起こししたテキストを OpenAI のテキスト生成 API（gpt-4o）へ送信し、演奏区間の推定、および会話の議事録・雑談への整理を行います。

送信されるのは録音音声と、そこから生成された文字起こしテキストです。送信されたデータの OpenAI 側での取り扱いについては、OpenAI のポリシーに従います（OpenAI は API 経由で送信されたデータを、原則として自社モデルの学習に利用しない旨を表明しています）。本アプリは AI による演奏への評価・採点は行わず、文字起こしと整理のみを行います。

4. アプリ内課金について

本アプリには、開発を支援するための任意の支払い（アプリ内課金）機能があります。支払いは Apple または Google の決済システムを通じて行われ、当方が決済情報（カード番号等）を取得・保管することはありません。この支払いは任意であり、支払いの有無によって利用できる機能が変わることはありません。

5. 情報の保存期間

利用者のアカウント情報・プロフィール・録音音声・コメント・予定などのデータは、利用者がアカウントを削除するまで、または当方がサービスの提供に必要な期間、保存します。アカウントを削除した場合の取り扱いは次項のとおりです。

6. 利用者の権利（データの削除）

利用者は、本アプリ内の「アカウント削除」機能から、いつでも自身のアカウントを削除できます。アカウントを削除すると、認証情報・プロフィールなど利用者個人に紐づくデータは削除されます。

ただし、利用者がグループ内で共有した録音・コメント・メモなどの共有資産は、他のメンバーの記録を保つため削除されず、作成者情報を切り離して（匿名化して）グループ内に残ります。これらの共有資産の個別削除を希望する場合は、第8項の問い合わせ先までご連絡ください。

  • アップロードするコンテンツについて、著作権その他の権利を侵害しないよう、自身の責任において確認してください。
  • 本アプリはクローズドなグループ内での練習記録の共有を目的としており、著作物の不特定多数への配布を意図するものではありません。個人の練習目的での利用の範囲であっても、著作権法および関連法令を遵守してください。
  • 当方は、利用者がアップロードしたコンテンツの著作権侵害について責任を負いません。

著作権者または権利者の方で、本アプリ上でご自身の著作物が無断利用されていると思われる場合は、第9項の問い合わせ窓口までご連絡ください。
